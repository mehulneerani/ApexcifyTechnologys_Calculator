# Gold & Black Scientific Calculator

## Project Overview

This project is a **Gold & Black themed Scientific Calculator** built using **HTML, CSS, and JavaScript**.
The calculator performs basic arithmetic operations while also including additional scientific features and keyboard support.

The interface is designed with a **luxury gold gradient background** and a **floating calculator panel** to create a modern and elegant user experience.

This project was created as part of the **Frontend Development Internship Task at Apexcify Technologys**.

---

## Features

• Basic arithmetic operations

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)

• Scientific functions

* Square (x²)
* Percentage (%)
* Logarithm (log)
* Exponent (^)
* Parentheses ( )

• User Interface

* Large display screen
* Responsive button grid layout
* Clear button and backspace functionality
* Gold and black themed design

• User Interaction

* Mouse click support
* Keyboard input support
* Real-time calculation display

• History Feature

* Users can open the history panel to view previous calculations.

## How It Works

1. Users enter numbers using the calculator buttons or keyboard.
2. The input is displayed on the calculator screen.
3. Arithmetic or scientific operations can be applied.
4. Pressing the **equals (=)** button calculates the result.
5. The **C button** clears the display and resets the calculator.
